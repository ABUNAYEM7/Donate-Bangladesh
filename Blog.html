Importanat FAQ

What is the Document Object Model (DOM)

DOM (Document Object Model) is a programming interface for web documents, representing the structure of a webpage.

How do you select element from dom

Select DOM elements using getElementById, getElementsByClassName, getElementsByTagName, querySelector, or querySelectorAll.

what is event delegation in the context of DOM, and why it is useful

Event delegation attaches a single event listener to a parent element, handling events for child elements. It's useful for improving performance, managing dynamic content, and simplifying code.

How do you manipulate an element's attributes and styles using DOM

Manipulate attributes with element.setAttribute() and styles with element.style.propertyName = 'value'.
